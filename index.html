L1 CRUD — Dict 操作

Dict 基本

d = {}                          # 建空 dict
d[key] = value                  # 加/改
del d[key]                      # 刪
if key in d:                    # check 存在
d.get(key)                      # 搵唔到 return None 唔會 crash
d.get(key, default)             # 搵唔到 return default
d[key]["field"]                 # nested dict
for k, v in d.items():          # loop，k=key, v=value
❌ d.balance — dot 係 class 嘅寫法
✅ d["balance"] — dict 用 bracket

❌ d["account_id"] — 加引號搵 literal string "account_id"
✅ d[account_id] — 唔加引號用 variable 嘅值
Dict 係 pointer，int 係 copy：
acc = self.accounts[id] → acc 指住同一個 dict，改 acc 會改原本
t_b = acc["balance"] → copy 咗個數字，改 t_b 唔影響原本

List 基本

lst = []                # 建空 list
lst.append(x)           # 加去尾
lst[0]                  # 第一個
lst[-1]                 # 最後一個
lst[:n]                 # 頭 n 個（n 太大唔會 error）
len(lst)                # 長度

Lazy Processing Helper（所有題都用，背住）

唔會自動到期。冇真正嘅 clock 行緊，唯一知道「而家幾點」就係有人 call method 帶 timestamp 入嚟。

def _process_cashbacks(self, timestamp):
    for pid, p in self.payment.items():
        if not p["received"] and timestamp >= p["cashback_time"]:
            self.accounts[p["account_id"]]["balance"] += p["cashback_amount"]
            p["received"] = True

# 所有 method 第一行（包括 create_account）：
self._process_cashbacks(timestamp)

L5-6 Concurrency

概念 1：async + await = 「可以暫停等」

# 普通：企喺度等
def cook_rice():
    wait_20_minutes()
    return rice

# Async：等嘅時候可以做其他嘢
async def cook_rice():
    await wait_20_minutes()
    return rice

加兩個字。async 喺 def 前面，await 喺要等嘅嘢前面。

概念 2：asyncio.gather = 「幾樣嘢一齊做」

# 排隊（35 分鐘）
rice = await cook_rice()       # 等 20 分鐘
soup = await cook_soup()       # 再等 15 分鐘

# 同時（20 分鐘）
rice, soup = await asyncio.gather(cook_rice(), cook_soup())

概念 3：Lock = 「一次一個人用」

lock = asyncio.Lock()
async with lock:
    acc["balance"] += amount    # 一次一個人行

L5 Batch Processing（完整 example，灰色部分因應題目改）

固定寫法。三步：1. 定義 execute_op 2. gather 同時跑 3. return list(results)

# === __init__ 加 ===
self.account_locks = defaultdict(asyncio.Lock)

# === 完整 example ===
async def process_batch(self, timestamp, operations):   # ← 固定
    self._process_cashbacks(timestamp)                     # ← 固定

    async def execute_op(op):                            # ← 固定
        aid = op.get("account_id") or op.get("source_id")  # ← 固定
        lock = self.account_locks[aid]                     # ← 固定
        async with lock:                                  # ← 固定
            # -------- 以下因應題目改 --------
            if op["type"] == "deposit":
                return self.deposit(timestamp, op["account_id"], op["amount"])
            elif op["type"] == "transfer":
                return self.transfer(timestamp, op["source_id"], op["target_id"], op["amount"])
            elif op["type"] == "pay":
                return self.pay(timestamp, op["account_id"], op["amount"])
            # -------- 以上因應題目改 --------

    results = await asyncio.gather(*[execute_op(op) for op in operations])  # ← 固定
    return list(results)                                # ← 固定

L6 Rate Limited（完整 example，灰色部分因應題目改）

同 L5 一樣嘅 skeleton，加 Semaphore 限制同時數量。Lock 包住扣錢，Sem 包住外部 call。

async def process_external_transfers(self, timestamp, transfers, max_concurrent):  # ← 固定
    self._process_cashbacks(timestamp)                     # ← 固定
    sem = asyncio.Semaphore(max_concurrent)                # ← 固定

    async def do_transfer(t):                            # ← 固定
        aid = t["account_id"]
        lock = self.account_locks[aid]                     # ← 固定
        async with lock:                                  # ← 固定：鎖住 account
            # -------- 以下因應題目改（check + 扣錢） --------
            if aid not in self.accounts:
                return False
            if self.accounts[aid]["balance"] < t["amount"]:
                return False
            self.accounts[aid]["balance"] -= t["amount"]
            self.accounts[aid]["outgoing"] += t["amount"]
            self.accounts[aid]["history"].append((timestamp, self.accounts[aid]["balance"]))
            # -------- 以上因應題目改 --------
        async with sem:                                  # ← 固定：限制同時數量
            await asyncio.sleep(0.01)                      # ← 固定：模擬外部 call
        return True

    results = await asyncio.gather(*[do_transfer(t) for t in transfers])  # ← 固定
    return list(results)                                # ← 固定
L5 vs L6 嘅分別：
L5：gather + Lock（全部同時跑，鎖住 account）
L6：gather + Lock + Semaphore（限制同時幾多個，鎖住 account + 模擬外部 call）

記住：Lock 同 Sem 分開用。Lock 包住改 data 嘅部分，Sem 包住外部 call 嘅部分。唔好 return 太早。

! Gotchas（做過嘅錯誤）

❌ _init_ 一個底線 → ✅ __init__ 兩個底線
❌ def method(account_id): 漏 self → ✅ def method(self, account_id):
❌ d.balance dot → ✅ d["balance"] bracket
❌ accounts.append(acc) → ✅ accounts[key] = acc dict 用 key
❌ balance <= amount return None → ✅ balance < amount 等於夠錢要畀轉
❌ for x in dict: x.key → ✅ x 已經係 key string
❌ payment = d[pid]: 多咗冒號 → ✅ 唔係 function 唔使冒號
❌ return IN_PROGRESS → ✅ return "IN_PROGRESS" 要引號
❌ acc.amount = amount 覆蓋 → ✅ acc["balance"] += amount 累加
❌ 邊 iterate 邊 delete dict → ✅ 先收集 keys 再 delete
❌ copy_ref = original 同一個 object → ✅ copy.deepcopy(original)
